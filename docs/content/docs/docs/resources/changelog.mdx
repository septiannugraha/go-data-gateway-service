# Changelog

## Version 1.2.0 (2024-01-15)

### New Features
- Added batch operations support for Tender endpoints
- Introduced webhook events for contract management
- New analytics endpoints for RUP
- GraphQL support (beta)

### Improvements
- Improved response time by 40%
- Enhanced error messages with more context
- Better pagination support with cursor-based navigation
- Optimized database queries for vendor performance

### Bug Fixes
- Fixed issue with date parsing in tender endpoints
- Resolved timeout errors in large data exports
- Fixed authentication token refresh logic

### Breaking Changes
- Deprecated `/api/v1/tenders` in favor of `/api/v1/tender/paket`
- Changed response format for error messages

### New Features
