# Tender & Non Tender API

## Pengenalan

Layanan Tender & Non Tender adalah bagian dari INAPROC API Gateway untuk memfasilitasi proses pengadaan barang/jasa pemerintah secara elektronik, mencakup tender dan non tender.

## Fitur Utama

- **E-Tendering**: Proses tender elektronik yang transparan
- **Non Tender**: Pengadaan di luar mekanisme tender
- **E-Purchasing**: Pembelian melalui e-katalog
- **Monitoring**: Dashboard monitoring real-time

## Base URL

```
Production: https://api.inaproc.id/v1
Sandbox: https://sandbox-api.inaproc.id/v1
```

## Autentikasi

Semua request harus menyertakan access token pada header Authorization.

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```
